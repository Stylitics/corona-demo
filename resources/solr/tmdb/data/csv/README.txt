Summary
=======

This dataset was composed using The Movie DB API, and few MovieLens datasets:
- `tmdb_5000_movies.csv` contains metadata for top 5000 movies of TMDB.
- `tmdb_5000_credits.csv` contains credits for top 5000 movies of TMDB.
- `ratings.csv` and users.csv contains 1 million ratings from 6000 users (with profiles) on 4000 movies. Imported from https://grouplens.org/datasets/movielens/1m/
- `links.csv` contains correspondence of MovieLens ID with TMDB ID and IMDB ID for 46000 movies. Imported from https://grouplens.org/datasets/movielens/20m/
